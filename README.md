# Iris Dataset Exploratory Data Analysis

This project demonstrates basic Exploratory Data Analysis (EDA) on the Iris dataset using Python.

## Features

* Load the Iris dataset from scikit-learn
* Convert target labels into species names
* Create scatter plots for petal and sepal measurements
* Generate a correlation heatmap
* Split data into training and testing sets
* Standardize numerical features using StandardScaler

## Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Installation

```bash
pip install -r requirements.txt
```

## Dataset

The Iris dataset is included in scikit-learn and contains measurements of three iris species:

* Setosa
* Versicolor
* Virginica
